#### **二.程序简介**
如上所说，该程序只是简单的类跳一跳小游戏，目前设有<font color='gree'>*Easy</font> <font color='blue'>   Normal</font>   <font color='red'>Hard*</font> 三种难度。
|难度|得分（分/格）|判定|区块大小|
|:--:|:--:|:--:|:-:|
|<font color='gree'>Easy|1|宽松|较大|
|<font color='blue'>Normal|2|正常|正常|
|<font color='red'>Hard|3|严格|较小|

游戏内玩家需要控制2D的小球，长按鼠标左键控制跳跃长度，跳跃至下一个平台上获得更高的分数，掉下平台即为失败。

游戏中的小球运动具有惯性，游戏外提供了最高分数记录以及重置最高纪录的功能。
